# Website

This website is built using [Docusaurus 2](https://docusaurus.io/), a modern static website generator.

## Installation

```console
npm install
```

## Local Development

```console
npm start -- --port 3000
```

## Update

```console
npx docusaurus --version
npm update
```

## Build & Deployment

```
npm run build
python3 transfer.py
```
